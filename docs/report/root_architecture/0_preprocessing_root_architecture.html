7.1 Contexte

Seedlings were grown in RhizoTubes¬Æ filled with sandy soil, allowing the visualization of the root system Jeudy et al. (2016)

The purpose of this script is to synthesize all my architecture data, to write all the import steps with the multiple specificities of the import. At the end, I will generate csv files of root architecture. Each file corresponds to a different type of data. For example, data per plant, per depth in the rhizotube etc.
